.. _myquick:

=================
Quick start guide
=================

Install Microprobe on your system by executing::

   > pip install --pre -U microprobe_all

Check the `pip` command documentation for different intallation
options. 

.. note::

    If you plan to contribute to the project, you should install
    Microprobe using the instructions here: :doc:`start_install_git`

Check :doc:`start_update` for more details on how to upgrade your installation.
