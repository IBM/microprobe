========
Examples
========

The following sections provide some examples on how to use Microprobe for 
the different targets implemented. Although we have split the examples by 
architecture, the concepts we introduce in each of these sections are common in 
all the architectures. Let us know which examples could be useful
for new users, so that we can extend the current set of examples.
